# Запускаем ассистента на базе OpenHands + Mistral LLMка

Предлагаю в качестве базы проект OpenHands (ребята уже какое-то время его разрабатывают и он опенсурсный + недавно релизнулись)

https://github.com/All-Hands-AI/OpenHands

Запускать будем Docker готовые образы.

В качестве LLMки выберем MistralAI Codestral

## Windows

Сначал установить Docker: 

скачать тут https://docs.docker.com/desktop/setup/install/windows-install/
(Docker Desktop for Windows - x86_64)

установить. У вас должна быть включена поддержка виртуализации на cpu. 
После установи желательно перезагрузиться
